How it works

Built into every save.
Read by every agent.

Ripple runs inside VS Code, maintaining a live dependency graph. Every file save updates it in real time.

Risk classification

Agents know before
they touch anything.

Every file is classified by blast radius. Agents read the risk level from the focus file and follow the WORKFLOW.md protocol accordingly — automatically.

● Dangerous
5+ importers or 15+ saves. Agent stops and asks for explicit confirmation before making any change. Cannot be overridden without user approval.
● Caution
2-4 importers or 8+ saves. Agent proceeds but verifies every caller still works after each change. Reports what it touched.
● Safe
0-1 importers, low churn. Agent proceeds directly. Low blast radius means changes are contained and safe to make without confirmation.
