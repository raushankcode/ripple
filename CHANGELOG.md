# Changelog

All notable changes to Ripple are documented here.

## [1.0.1] — 2025-01-05

### Fixed

- Auto-sync WORKFLOW.md to CLAUDE.md and AGENTS.md on every save
- Fixed generateWorkflow closing brace bug causing TypeScript compile errors
- Added Machine Learning category for marketplace discoverability

### Changed

- Marketplace icon updated
- Package description updated with Context Rot framing
- Display name updated to "Ripple — Live Architectural Intelligence"

## [1.0.0] — 2025-01-04

### Added

- Initial release
- Impact Lens sidebar — live dependency graph for every file
- CodeLens caller counts above every function declaration
- Safety Check — blast radius warnings before git commit
- Copy Agent Prompt — right-click any TypeScript file
- WORKFLOW.md generation — live AI agent protocol on every save
- Focus files — 200-token per-file architectural snapshots
- Support for Next.js, Vite, React Router, Turborepo, MobX,
  React Query, tRPC, Prisma, Tailwind
- history.json — permanent architectural memory
- context.json, context.files.json, context.symbols.json
