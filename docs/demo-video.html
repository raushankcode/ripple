Privacy model

Local-first context, owned by the project.

Ripple is not a cloud index. It generates project context inside the workspace and stores saved history with project-relative paths.

No telemetry No account required and no hidden analytics pipeline.
No code upload Analysis runs locally inside the VS Code workspace.
Portable history Saved context uses paths like src/graph.ts, not machine paths.
Engineering signal Ripple complements tests, review, and judgment. It does not replace them.
