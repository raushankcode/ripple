Risk classification

Agents know before
they touch anything.

Every file is classified by blast radius in real time. Agents read the risk from the focus file and follow the auto-generated WORKFLOW.md protocol accordingly — on every task, automatically.

● Dangerous
5+ importers or 15+ saves. Agent stops and asks for explicit confirmation before making any change. Cannot be bypassed without user approval.
● Caution
2–4 importers or 8+ saves. Agent proceeds but verifies every caller still works after each change. Reports every file it touched.
● Safe
0–1 importers, low churn. Agent proceeds directly. Low blast radius means changes are contained and safe to make without confirmation.
